RouYi-Cloud

若依,官方demo,文档,下载;

项目部署

环境

nacos

mysql

redis

后端

  1. git clone,idea打开项目,Encoding都用UTF-8.
  2. 创建数据库ry-cloud并导入数据脚本ry_2020520.sql（必须），quartz.sql（可选）
  3. 创建数据库ry-config并导入数据脚本ry_config.sql（必须）其中配置(nacos,mysql,redis)需要修改
  4. 配置nacos持久化，修改conf/application.properties文件，增加支持mysql数据源配置
  1. 打开运行基础模块（启动没有先后顺序）
  • RuoYiGatewayApplication （网关模块 必须）
  • RuoYiAuthApplication （认证模块 必须）
  • RuoYiSystemApplication （系统模块 必须）
  • RuoYiMonitorApplication （监控中心 可选）
  • RuoYiGenApplication （代码生成 可选）
  • RuoYiJobApplication （定时任务 可选）
