学习游泳

入门先学蛙泳腿(手随意,都能漂)，再学自由泳(自由泳要规范避免肩膀受伤)。参考B站梦觉教练

[TOC]

蛙泳

动作可以分解成手部动作、腿部动作和换气。

手部动作

对于新手动作幅度不要太大，手始终在胸前，主要是确保能够换到期，每次划手配合抬头，如果还是换不到气，就两蹬一划，确保足够的升力。

动作要领：
  1. 起:两个手臂自然前伸,手掌张开,掌心向下,与水平面平行,身体处于自然伸直状态。
  2. 划:手肘伸直，掌心由向下慢慢转为向外，手掌倾斜大约45度，边转手掌边将全臂向外斜下方推开，这时是没有向前的推进作用的，不需要太用力，以免浪费体力。
  3. 抱:高肘用力抱住更多的水，然后向后推。
  4. 收:双手从胸前向前伸直，回到起的状态。

腿部动作

动作要领：
  1. 收:腿伸直，脚绷直。
  2. 翻:膝盖弯曲，小腿上抬，大腿不要弯曲太多，避免不必要的阻力；双脚勾起，两腿保持60度夹角，预备势能，准备蹬腿。
  3. 蹬:双脚勾紧，增大蹬水面积，向后踢腿。
  4. 夹:蹬腿后期，双腿迅速并拢提供推力，双脚绷直(在膝盖承受范围内尽量绷直)，回到收的状态。

蛙泳换气

在蹬腿划手后有足够的升力，抬头换气，抬头不要太猛，平缓的抬头有更多的换气时间。有些人游抬头蛙,是脚向下蹬水,就可以保持头不沉.

自由泳

入门采用前交叉两次打退，三划一换气(避免单侧换气,肩部受伤)。身体配合划手摆动，推水时吐气，移臂时侧身转头换气.不换气时,头保持三分一(后脑勺)出水面,保持平稳,如果左右晃的厉害,要检查打腿和划水的姿势有没有问题.

手部动作

初学最好采用前交叉换手：在划水的那一只手运动时，另一只手保持在前方紧贴头部，以减少阻力；在划水完毕，移臂回到前方时，另一只手才开始下一次划水。

入水：入水时手掌的理想的角度是45度角，要由指尖先入水(不要大拇指入水,会对肩膀产生不必要的压力),也就是说手掌朝向外侧(45度)的角度进入,手臂入水后的第一个动作都该是前伸而不是马上发力下压。臂入水的顺序是:手 →前臂→上臂。

抱水：在手臂前伸到最远处,前臂向下弯曲(避免直臂压水),使前臂和手掌与前进方向保持90度。(过程中要避免过度的压水,只会使上半身抬起,腿下沉,增加阻力)

推水：向后快速推水。手掌与前进方向保持90度(增大受力面积,这是自由泳的主要动力)。不能拖肘(是向后用力而不是向下),不可过中线(避免左右摇摆)。

出水：推水到大腿边,手肘先出水。

高肘移臂：配合侧身,手肘抬高避免阻力,手掌前伸到最远处入水。

腿部动作

入门打腿只是为了保持平衡，可以先夹板练习。要省力就二次腿,要是腿会沉就六次腿(二次腿的基础上加上一个略轻的交叉腿),专业长距离选手都是在不同模式之间切换的,感觉腿要沉就加一个交叉腿.其实不打腿也是可以游的,只是阻力比较大,好的打腿可以方便转体,提高腿部浮力,以减少阻力.

要点：

1.自由泳腿部的打水动作,虽然也有一定的推进作用,但主要是起到保持身体的平衡。

2.膝盖自然弯曲,但不能弯曲太大,这样的话小腿打水就显得不够力,而且加大了前进的水阻力,看不到打水时的水花;如果绷紧,伸得太直,会让腿部肌肉过于紧张,太累,而且水花也大,最佳状态是弯曲160°。

3.尽量放松脚踝,绷直脚尖,大腿带动小腿打水,腰部以下用力,上半身保持不动,这样打水才有力度,而且不会左右晃动。

换气

可以采用三划一换,在推水结束开始移臂的时候配合转体的时候转头换气,怕进水可以在抱水时轻微压水配合抬头,游快了就不需要压水和抬头了.(压水会对肩膀产生而外压力,不可养成压水换气的习惯)

节奏

尽量采用前交叉,在每次划水时候另一只手伸向前方保持不动,划水时配合打腿,打腿主要是保持腿不沉,主要是二次推,在感觉腿下沉的时候补一次交叉腿,在划水完毕空中移臂的时候可以漂一会,不容易累.

水中练习

中心前移

将重心控制在前部,避免腿下沉

两边换气

三划一换,前置手前伸,脸贴着肩,转体吐气转头换气

陆上练习

力量训练

俯卧撑(波比跳),引体向上,悬臂举腿

拉力带

柔韧训练

正反手压肩,压脚踝

纠正练习

转肩甩手臂练习:直臂,不要过中线

避免移臂时小手臂先动:

  1. 移臂时注意手指朝向水面;
  2. 转体核心保存稳定,避免左右扭

固定移臂入水点,避免过中线:

前伸转体换气练习:

  1. 前置手延展伸直(转体换气使,肩部应该贴着脸)
  2. 头顶朝前,先转体,后转头,吸气,头转回去身体再转回去
