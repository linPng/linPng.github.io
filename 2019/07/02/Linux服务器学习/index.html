常用的目录

参考

/bin (binary)是二进制执行文件目录，主要用于具体应用

/sbin (system binary)是系统管理员专用的二进制代码存放目录，主要用于系统管理

/usr (unix system resource)unix系统资源文件目录,该目录经常用于安装各种软件

/dev (device)系统硬件设备目录

/home (home)家目录

/proc (process)内存映射目录

/var (variable)该目录存储的文件经常会发生变化

/boot 系统启动核心目录，用于储存系统启动文件

/etc 系统主要配置文件目录

/lib (library)系统资源文件类库目录

/mnt 存放临时的映射文件系统，我们常把软驱和光驱挂装在这里的floppy和cdrom子目录下

/tmp 存放临时文件的目录

vim的使用

参考

命令模式

  • i 切换到输入模式，以输入字符。
  • : 切换到底线命令模式，以在最底一行输入命令。
  • dd 剪切
  • p 复制
  • GG 跳转到页末
  • gg 跳转到页首
  • 数字+gg 跳转到对应行
  • $ (shift+4) 跳转到行末
  • 0 跳转到行首

输入模式

  • ESC，退出输入模式，切换到命令模式

底线命令模式

  • set nu/set nonu 显示/取消显示行号
  • /xx 搜索，n下一项，N上一项
  • q 退出程序
  • w 保存文件
  • q! 若曾修改过档案，又不想储存，使用 ! 为强制离开不储存档案。
