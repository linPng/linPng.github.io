fastDFS的学习

简介

FastDFS是一个开源的轻量级分布式文件系统。参考资料1

安装

参考资料2

1.下载

cd ../usr/fastDFS

git clone https://github.com/happyfish100/fastdfs.git

git clone https://github.com/happyfish100/libfastcommon.git

git clone https://github.com/happyfish100/fastdfs-nginx-module.git

安装libevent

yum -y install libevent

安装libfastcommon

cd libfastcommon/

./make.sh

./make.sh install

cp /usr/lib64/libfastcommon.so /usr/lib

安装Tracker服务

cd ../fastdfs

./make.sh

./make.sh install

cp /usr/fastDFS/fastdfs/conf/* /etc/fdfs

vim /etc/fdfs/tracker.conf

//修改配置base_path=/home/yuqing/FastDFS->base_path=/home/FastDFS

mkdir /home/FastDFS //如果目录不存在

/usr/bin/fdfs_trackerd /etc/fdfs/tracker.conf restart //启动

vim /etc/rc.d/rc.local //末行加上 /usr/bin/fdfs_trackerd /etc/fdfs/tracker.conf restart

安装storage服务

vim /etc/fdfs/storage.conf

修改了以下内容：

/usr/bin/fdfs_storaged /etc/fdfs/storage.conf restart //启动

vim /etc/rc.d/rc.local //末行加上/usr/bin/fdfs_storaged /etc/fdfs/storage.conf restart

测试

/usr/bin/fdfs_test /etc/fdfs/client.conf upload /home/test/test.jpg

报错：

ERROR - file: ../client/client_func.c, line: 257, “/home/yuqing/fastdfs” can’t be accessed, error info: No such file or directory

修改：

vim /etc/fdfs/client.conf

报错：

ERROR - file: tracker_proto.c, line: 49, server: 172.18.224.114:22122, response status 2 != 0
ERROR - file: ../client/tracker_client.c, line: 1077, fdfs_recv_response fail, result: 2
ERROR - file: tracker_proto.c, line: 49, server: 172.18.224.114:22122, response status 2 != 0
ERROR - file: ../client/tracker_client.c, line: 899, fdfs_recv_response fail, result: 2
tracker_query_storage fail, error no: 2, error info: No such file or directory

修改：vim /etc/fdfs/tracker.conf 配置allow_hosts

结果

example file url: http://49.235.153.134/group1/M00/00/00/rBEADl1uK92AMU4kABB9d1VCjF8914.jpg

FastDFS 和nginx整合

在Storage上安装nginx，或给Nginx添加 FastDFS 模块。

cd /usr/fastDFS/fastdfs-nginx-module/src/

vim config //去掉其中的/local

cp mod_fastdfs.conf /etc/fdfs/

vim /etc/fdfs/mod_fastdfs.conf

修改了以下内容：

cp /usr/lib64/libfdfsclient.so /usr/lib/ //可能已存在

mkdir -p /var/temp/nginx/client //创建nginx/client目录

安装nginx

./configure –prefix=/usr/nginx –with-http_stub_status_module –with-http_ssl_module –pid-path=/var/run/nginx/nginx.pid –lock-path=/var/lock/nginx.lock –error-log-path=/var/log/nginx/error.log –http-log-path=/var/log/nginx/access.log –with-http_gzip_static_module –http-client-body-temp-path=/var/temp/nginx/client –http-proxy-temp-path=/var/temp/nginx/proxy –http-fastcgi-temp-path=/var/temp/nginx/fastcgi –http-uwsgi-temp-path=/var/temp/nginx/uwsgi –http-scgi-temp-path=/var/temp/nginx/scgi –add-module=/usr/fastDFS/fastdfs-nginx-module/src

make

make install

配置nginx

server {
listen 80;
server_name localhost;#自己的地址
location /group1/M00/{
#root/home/FastDFS/fdfs_storage/data;
ngx_fastdfs_module;
}
}
