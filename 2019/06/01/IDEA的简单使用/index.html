IDEA的简单使用

心得
发布日期:   2019-06-01

简介

工具用得好，下班下得早。IntelliJ IDEA真的好用啊，所以很有必要花点时间深入学习一下如何使用idea。

快捷键

Ctrl+Shift+N//快速找file,不带shift是针对classes,还有all、symbols、actions

Ctrl+Shift+E//打开最近修改文件列表，不带shift是打开最近打开文件列表

Alt+Shift+C//最近的更改

Ctrl+F12//显示当前文件结构

Ctrl+H//显示类结构图（类的继承层次）

Ctrl+Shift+Space//自动补全代码

Ctrl+Shift+Backspace//跳转到上次编辑的地方

Ctrl+Alt+left/right//返回至上次浏览的位置

Alt+left/right//切换代码视图

Alt+Up/Down//在方法间快速移动定位

Ctrl+Shift+Up/Down，向上/下移动代码块

Ctrl+Shift+U//大小写转化

Ctrl+Alt+T//把代码块包起来，if、try之类的

Ctrl+Alt+J//用动态模板环绕

Ctrl+click//shift再点文件列表可以打开文件所在位置或文件，对变量使用是跳转到定义处

Alt+Shift+Up/Down，上/下移一行

Ctrl+Alt+L//格式化代码

Alt+F1//查找代码所在位置

Ctrl+Shift+V/Insert//打开粘贴板

Alt+Insert//打开构造器

Shift+Click//shift再点文件列表可以关掉它，等于点那个X

Ctrl+W//可以选择单词继而语句继而行继而函数,+shift则相反

Ctrl+[ 或]//去找最近的{或}

Ctrl+O//重写方法

Ctrl+I//实现方法

Ctrl+Y//删除行

Ctrl+X//剪切行

Ctrl+C//复制行

Ctrl+D//复制黏贴行

Shift Shift//文件参数搜索

Ctrl+Alt+B//打开时实现类

Alt+F7//查找方法被调用的位置

Ctrl+shift+T//跳转或创建测试用例

模版

IDEA database

IDEA database不仅可以用来查看数据关系，还可以用scripted Extensions来逆向生成SQL。

  1. 在idea右侧菜单栏Database中添加Data Source,选择需要的Schemas
  2. 找到对应的表,右键Scripted Extensions中选择生成脚本
  3. 如果没有脚本,则Go to Scripts Directory创建脚本
  4. 如果不想写脚本,就安装Free Mybatis plugin插件,mybatis-generator

Plugins

常用的插件有:

  • Alibaba Java Coding Guidelines:帮助校验代码,使代码更规范
  • CamelCase:切换大小驼峰或者用_-分隔单词.(Shift + Alt + U)
  • CodeGlance:滚动框代码预览
  • Codota:智能代码提示
  • Convert YAML and Properties File:转换YAML和Properties文件
  • Free Mybatis plugin:1.mapper跳转xml;2.mybatis-generator
  • JPA Support:也是逆向代码
  • GenerateAllSetter:快速生成setter方法(Alt+Enter)
  • GsonFormat:通过json串快速生成实体(Alt+Insert)
  • GsonFormatPlus:升级版本
  • Maven Helper:pom文件增加Dependency Analyzer窗口,查看解决冲突,以列表和树形式查看依赖
  • nginx Support/NodeJS/vue.js:支持在idea启动nginx/NodeJS/vue.js
  • SequenceDiagram:右键方法生成时序图
  • Grep Console:过滤日志
  • JRebel and XRebel for IntelliJ:热部署 需要收费 不推荐
  • Statistic:项目文件统计 旧版本idea要下载旧版本插件
  • .ignore:忽略文件提交
  • SonarLint:比阿里的更细节
  • CheckStyle-IDEA:比阿里和SonarLint还细节

Postfix

和Live Template类似,利用.key来快速写代码.

在Setting>Editor>General>Postfix Completion里有可以添加模版,默认自带了java,sql,js的常用模版.

Debug

开启debug，可以在断点中设置条件；

蓝色折线箭头+灰色按钮=单行执行（不会进入）

蓝色直线箭头+灰色按钮=单布进入（只会进入自定义方法）

红色直线箭头+灰色按钮=单步进入（也会进入系统方法）

蓝色反箭头+灰色按钮=单步跳出

社区版

下载社区版,有遇到的问题（在新机上安装）

  1. svn关联,在setting中svn设置本地ToroiseSVN(还是ToroiseSvn顺手)
  2. maven关联,直接download重新下载依赖,如果下载慢可以在maven的setting.xml(idea中选pom文件右键可以快速创建或打开)中加镜像,也可以提前拷贝旧电脑的maven仓库；可以下载旧版本maven
    1. Unresolved dependency: ‘jdk.tools:jdk.tools:jar:1.6’ 可能是下载的jkd的lib下没有tools.jar(jdk8)
  3. 快捷键发现crtl+y不是删除而是取消撤回，社区版默认的keymap是WindowsPorperRedo
