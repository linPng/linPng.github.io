数据类型

Python中的变量不需要声明。每个变量在使用前都必须赋值，变量赋值以后该变量才会被创建。

在Python中，变量就是变量，它没有类型，我们所说的”类型”是变量所指的内存中对象的类型。

  • Numbers（数字）:int、float、bool、complex（复数）。
    • 支持运算符:+ - * / ** % //
  • String（字符串）
    • 反斜杠可以用来转义，使用r可以让反斜杠不发生转义。
    • 字符串可以用+运算符连接在一起，用*运算符重复。
    • Python中的字符串有两种索引方式，从左往右以0开始，从右往左以-1开始。
    • Python中的字符串不能改变。
    • 三对引号表示多行字符串.
    • 通过在字符串前加 r 或 R表示自然字符串.
    • 字符串的截取的语法格式如下：变量 [头下标: 尾下标: 步长]
  • List（列表）是写在方括号之间、用逗号分隔开的元素列表。
  • Tuple（元组）与列表类似，不同之处在于元组的元素不能修改。元组写在小括号里，元素之间用逗号隔开。
  • Sets（集合）是一个无序不重复元素的集。可以使用大括号 或者 set()函数创建set集合，注意：创建一个空集合必须用 set() 而不是 { }.
  • Dictionaries（字典）是Python中另一个非常有用的内置数据类型。是一个无序的键 : 值对集合。关键字必须使用不可变类型，也就是说list和包含可变类型的tuple不能做关键字。在同一个字典中，关键字还必须互不相同。创建空字典使用{ }。

基础语法

  • 注释:单行注释以 # 开头，多行注释用三对单引号（’’’）或者三对双引号（”””）将注释括起来。
  • 标识符:对大小写敏感,由字母,数字和下划线组成组成,开头只能是字母或下划线.
  • 缩进:使用缩进来表示代码块,缩进的空格数是可变的,但是同一个代码块的语句必须包含相同的缩进空格数.
  • Python可以在同一行中使用多条语句，语句之间使用分号(;)分割.
  • 像if、while、def和class这样的复合语句，首行以关键字开始，以冒号( : )结束，该行之后的一行或多行代码构成代码组。
  • 使用import 与 from…import导入函数或模块
  • 复合赋值:a, b = 0, 1 ,同时负责ab
  • 迭代器:iter() 和 **next()**用于创建迭代器和访问下一个元素.
  • 循环:while条件循环while 判断条件：;for遍历循环for i in range(5):(range会生成数列)
  • 函数:使用def声明,括号里写入参,入参可用等式赋默认值,不用指定返回类型
  • 异常:使用try和except,except可以多个,都不满足则返回上层报错,最后一个except子句可以忽略异常的名称.
