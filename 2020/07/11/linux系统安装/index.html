linux简介

linux是一套免费使用和自由传播的类UNIX操作系统,常见的发行版本有RedHat,Centos等.由于红帽被IBM收购,可以多尝试其他的linux发行版,如:华为的openEuler.

linux安装准备

去Centos官网下载最新的centos8.2:CentOS-8.2.2004-x86_64-dvd1.iso

或者 去openEuler下载openEuler-21.09-everything-x86_64-dvd.iso

使用UltraISO制作启动盘

官网下载并安装UltraISO，安装完成后点击试用。

点击文件，选择打开，找到Centos8包所在的文件夹，选择Centos8包。

插入准备好的U盘,打开目标ISO,选择U盘为本地目录,点击顶部菜单中的 启动选择写入硬盘映像（注意是硬盘不是软盘）

盘符名称为：CentOS-8x86

进入BIOS模式设置U盘启动

运行dxdiag，查看bios信息

BIOS：Default System BIOS

把U盘插在电脑上（最好是后面的USB口，有些电脑的前置USB口不支持U盘启动）

开机时按Delete，进入BIOS

进入Advanced BIOS Features ，Hard Disk Drives ，将U盘选为优先（-号操作）

F10保存并退出

安装CentOS

需要按TAB调整盘符名称

进入安装界面。

选择一下语言，和安装目的地，我是选择了带GUI的服务器。

安装目的设置选择自定义存储配置，手动分区。

点击开始安装，设置root密码，创建用户，等待半个小时就安装好了。

重启，进入BIOS修改硬盘优先，安装完成。

可能出现的问题

出现dracut:/#

默认盘符名称为CentOS-8-2-2004-x86_64-dvd

安装openEuler

和CentOS类似,要注意盘符名称

\isolinux\isolinux.cfg 中的LABEL=openEuler-21.09-x86_64,而实际盘符名称限制了长度,所以变成openEuler-2,在cfg中修改为正确的盘符名.
